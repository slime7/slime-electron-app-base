# slime-electron-app-base

快速创建一个 electron 应用。

## Project setup
```
git clone https://github.com/slime7/slime-electron-app-base.git
cd slime-electron-app-base
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
